Қарбалас уақытта автобус ішінде не болып жатыр?

Жолаушылар тығыздық туралы ақпарат болмаса, уақытын және жайлылығын жоғалтады.

🚌

Толып кету мәселесі

Қарбалас уақытта автобустар жолаушыларға сыймай кетеді. Бұл халықтың қолайсыздық сезінуіне және жанжалдарға себеп болады.

⏰

Уақыт шығыны

Жолаушылар толы автобусты күтпей немесе бос маршрутты таңдай алмай, орынсыз уақыт жоғалтады.

📉

Тиімсіз жоспарлау

Көлік компаниялары нақты деректерсіз қосымша автобустарды дер кезінде ұйымдастыра алмайды.

📵

Ақпарат жоқтығы

Жолаушыларда автобусқа мінбей тұрып, ішіндегі жағдайды білуге мүмкіндік болмайды.

// Жүйенің жұмысы

Деректер қалай жиналады?

Екі деректер көзін біріктіру арқылы жолаушылар санының дәл есебі жүргізіледі.

1

IR Сенсорлар

Есіктің үстіне орнатылған инфрақызыл сенсорлар кіріп-шыққан адамдарды санайды

2

Төлем Деректері

Банк карталары мен мобильді төлемдер арқылы жолаушылар саны нақтыланады

3

APC Жүйесі

Automated Passenger Counting жүйесі деректерді бірлестіріп, нақты санды есептейді

4

Бұлтты Өңдеу

Деректер серверге жіберіліп, нақты уақыт режимінде өңделеді

5

Карта Дисплейі

Пайдаланушы қосымшасында маршруттар түсті индикаторлармен белгіленеді

// Көрсетілім

Қолданылатын технологиялар

Заманауи бағдарламалық және аппараттық шешімдердің үйлесімі.

APC — Automated Passenger Counting

Инфрақызыл сенсорлар арқылы жолаушылардың кіріп-шығуын автоматты түрде санайтын жүйе. Қозғалыс бағытын да ажыратады.

Аппараттық жабдық

Real-time Деректер Трансляциясы

MQTT хаттамасы арқылы автобустан серверге деректер 5 секунд сайын жіберіледі. Жоғары сенімділік пен төмен кідіріс.

IoT / Желі

Карта API & Визуализация

OpenStreetMap негізінде маршруттар динамикалық түспен белгіленеді. Пайдаланушы картаны интерактивті түрде зерттей алады.

Фронтенд

Деректер Базасы & Backend

PostgreSQL + TimescaleDB уақыт белгілері бар деректерді тиімді сақтайды. REST API арқылы мобильді қосымшаға жіберіледі.

Backend

Машиналық Оқыту

Тарихи деректер негізінде ML алгоритмдері болашақ тығыздықты болжайды. Қарбалас уақыттар мен демалыс күндерінің өрнектерін үйренеді.

ML / AI

Мобильді Қосымша

Android & iOS платформаларында жұмыс істейтін қосымша маршрутты іздеуге, ескертулер алуға мүмкіндік береді.
